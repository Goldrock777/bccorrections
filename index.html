Capabilities
Everything ACD Requires

Built to meet all four criteria of BC Corrections' uniform program requirements.

🌐

Web-Based Ordering

Officers place orders through an online catalogue from any browser — no installed software required.

📦

Inventory Management

Real-time stock tracking across all 10 correctional centres with automated low-stock alerts and reorder triggers.

🏆

Point Allotment System

Each officer receives 500 annual points. Every uniform item has a point value. The system automatically tracks balances and enforces limits.

🚚

Centre-Based Delivery

All orders delivered directly to the officer's assigned correctional centre. Full shipment tracking from order to delivery.

📊

Analytics & Reporting

Administrators see order trends, centre activity, delivery performance, and compliance metrics on a live dashboard.

🔒

Secure & Auditable

Role-based access control, full audit trails, and data stored in-Canada ensure security and compliance.

📱

Mobile Responsive

Works seamlessly on phones, tablets, and desktop — officers can order from any device on or off duty.

⚡

Rapid Deployment

System is fully operational and can be provisioned for all 1,725 officers within days, not months.

Technical Specifications
Purpose-Built for BC Corrections

Designed to meet every operational requirement of the Adult Custody Division uniform program.

Uniform Catalogue

  • Uniform Shirt — 40 pts
  • Uniform Pants — 50 pts
  • Duty Boots — 120 pts
  • Outerwear Jacket — 100 pts
  • Duty Belt — 80 pts
  • Badge & Insignia — 20 pts
  • Tactical Holster — 85 pts
  • Tactical Gloves — 25 pts
  • Duty Cap — 30 pts
  • Long Sleeve Shirt — 45 pts

Correctional Centres

  • Surrey Pre-Trial Services Centre
  • North Fraser Pre-Trial Centre (Coquitlam)
  • Maple Ridge Correctional Centre
  • Vancouver Island Regional CC (Victoria)
  • Nanaimo Correctional Centre
  • Kamloops Regional CC
  • Prince George Regional CC
  • Chilliwack Youth Custody Centre
  • Oliver Correctional Centre
  • Okanagan Correctional Centre (Kelowna)

System Features

  • 500 annual points per officer
  • Automatic point balance tracking
  • Order status: Pending → Shipped → Delivered
  • Full points ledger per officer
  • Admin reorder management
  • Compliance & audit trail
  • SLA performance monitoring
  • Data persistence across sessions

Platform Requirements

  • No software installation required
  • Browser-based (Chrome, Firefox, Safari, Edge)
  • Mobile, tablet & desktop responsive
  • Dark mode support
  • No external dependencies
  • Data stored securely in-Canada
  • 99.9% uptime SLA
  • Role-based access control
